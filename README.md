# MELMA-W-Evaluation
MELMA-W is an open-source framework for automated evaluation of clinical LLM outputs. This repository contains the web application source code, anonymized evaluation data, and Python scripts used for statistical analysis and figure generation in the associated MELMA-W study.

## Data Availability

All anonymized evaluation data and MELMA-W reports used in this study are publicly available on Zenodo:  
DOI: https://doi.org/10.5281/zenodo.18436423
